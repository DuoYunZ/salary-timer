# 摸鱼时薪计算器

一个可爱的网页工具，帮助你计算和追踪工作时间与实际收益。

## 特点

- 🎯 实时计算时薪和收益
- 🐟 支持摸鱼时间记录
- 📅 自动保存每日数据
- 🦄 可爱的界面设计
- 🎠 萌趣的小毛驴动画

## 使用方法

1. 输入你的月薪和工作天数
2. 点击"确认"计算时薪
3. 点击"开始工作"开始计时
4. 摸鱼时点击"摸鱼"按钮
5. 查看右上角实时时薪变化
6. 点击"查看日历"回顾历史记录

## 在线使用

访问 https://[你的GitHub用户名].github.io/salary-timer/

## 本地使用

1. 下载本仓库
2. 直接打开 `index.html` 文件
3. 开始使用

## 技术栈

- HTML5
- CSS3
- JavaScript
- LocalStorage 用于数据存储
